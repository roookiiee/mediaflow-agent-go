# AI Product Metrics Skill

Use this skill when an Agent workflow needs measurable success criteria.

Metric checklist:

1. Experience: first response latency, end-to-end job duration, reviewer edit effort.
2. Success rate: valid output rate, retry rate, safety rejection rate, publish pass rate.
3. Cost: token usage, TTS seconds, fallback model usage, storage footprint.
4. Reliability: failed tool calls, timeout count, malformed output count.
5. Optimization loop: compare prompt/model/tool versions with the same evaluation cases.
