# Video Localization Skill

Use this skill when the user asks for translation, dubbing, subtitles, TTS, short-video localization, or multilingual content.

Workflow:

1. Identify target locale, channel, source script, desired voice style, and constraints.
2. Preserve intent and product claims; rewrite hooks to sound native for the target platform.
3. Keep TTS lines short, concrete, and easy to pronounce.
4. Generate subtitle-friendly segments with stable order.
5. Add voice-cloning approval requirements whenever cloned or celebrity-like voices are requested.
6. Return artifacts that a reviewer can edit directly: localized script, TTS plan, content pack, and quality metrics.
